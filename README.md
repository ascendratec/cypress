# 🚀 Curso de Cypress – Ascendra Tecnologia

Bem-vindo(a) ao repositório oficial do **Curso de Cypress da Ascendra**!  
Este projeto foi criado para apoiar o aprendizado prático de **testes automatizados end-to-end (E2E)** utilizando o **Cypress**, uma das ferramentas mais modernas e eficientes para testes de aplicações web.

---

## 📌 Objetivo do Repositório

Este repositório tem como objetivo:

- Servir como **base prática** para as aulas do curso  
- Centralizar **exemplos, exercícios e desafios**
- Demonstrar **boas práticas de automação de testes**
- Facilitar o acompanhamento da evolução do aluno ao longo do curso

---

## 🧠 Pré-requisitos

Para aproveitar melhor o curso, é recomendado que você tenha conhecimento básico em:

- JavaScript
- HTML e CSS
- Conceitos básicos de testes de software
- Linha de comando
- Git e GitHub (nível iniciante)

---

## 🛠️ Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)
- [Cypress](https://www.cypress.io/)
- JavaScript
- Git
- npm
